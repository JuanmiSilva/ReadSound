# Read&Sound

Aplicación multiplataforma para lectura de archivos EPUB con sincronización mediante CFI, búsqueda de texto y navegación avanzada.

## Características

- Lectura de archivos EPUB
- Navegación por capítulos
- Barra de progreso
- Búsqueda dentro del libro
- Captura de CFI al pulsar palabras
- Aplicación Electron (escritorio)
- Aplicación Android nativa
- Sistema preparado para sincronización de audio

---

## Tecnologías utilizadas

### Escritorio
- Electron

- Bootstrap

### Android
- Kotlin

## Instalación

### Electron

Entrar en la carpeta:

```bash
cd electron
```

Instalar dependencias:

```bash
npm install
```

Ejecutar:

```bash
npm start
```
